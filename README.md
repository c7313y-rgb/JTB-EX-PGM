# JTB教育事業部 探究プログラム営業デモアプリ

JTBの教育事業部営業が、学校訪問時に修学旅行組込型・通常授業組込型の企業共創プログラムを選び、概算見積を即時提示するための静的デモアプリです。

## 実装済み機能

- 2モード切替
  - ① 修学旅行組込型：タビマエ、東京での企業共創学習1日、タビアト構成
  - ② 通常授業組込型：企業、テーマ、実施コマ数、希望日から概算化
- 高品質デモビジュアル
  - 実写調の生成体験画像をヒーローと各プログラムに複数枚実装
  - 企業ロゴは公開配慮のためテキスト表現で掲載
- デモプログラム登録
  - 三越伊勢丹：LOCAL VALUE STUDIO / 高校生地域バイヤー体験
  - イトーキ：WORKPLACE FUTURE QUEST / 未来の学び場・働き方デザイン
  - カムジャバト × 三越伊勢丹：FIELD TO FLAGSHIP STORE / 畑から百貨店へ。カムジャバト探究プログラム
  - フレーベル館：AIリテラシー × AI絵本メーカー
  - GAMZA BATT / BATT JP：BRAND STUDIO / 商品企画とZ世代マーケティング
  - 日本気象協会：WEATHER DATA QUEST / 気象データで地域防災
- 条件検索
  - キーワード
  - 企業
  - テーマ
  - 対象学年
- 営業提案向けの詳細表示
  - 学習成果
  - 実施フロー
  - 体験シーン
  - 成果物
  - 営業メモ
- 概算見積自動生成
  - 受講人数
  - 希望日
  - 実施コマ数
  - タビマエ、タビアト数
  - 人数超過時の分散実施調整費
- PDF / 印刷
  - ブラウザの印刷機能でPDF保存可能
  - 印刷時は見積書中心のレイアウトに自動最適化
- JTB側サーバー送信デモ
  - 疑似送信ログ
  - JSONペイロード表示
  - JSON保存
- GitHub Pages対応
  - HTML / CSS / JSのみで動作
  - ビルド不要

## ローカル確認

- `index.html` をブラウザで直接開くか、任意の静的サーバーで配信してください。
- 例: `python3 -m http.server 4187`
- 印刷確認はブラウザの `PDF / 印刷` ボタンから行えます。

## 公開手順

1. GitHubで新規リポジトリを作成
   - 推奨名：`jtb-education-demo`
   - PublicまたはPrivateを選択
2. 本フォルダ内の全ファイルをアップロード
   - `index.html`
   - `styles.css`
   - `app.js`
   - `assets/`
   - `README.md`
3. GitHubの `Settings` → `Pages` を開く
4. `Build and deployment` の Source を `Deploy from a branch` に設定
5. Branch を `main`、folder を `/root` に設定
6. 保存後、数十秒〜数分で公開URLが発行されます

## 注意

- 本アプリは営業検証用デモです。
- 企業名、プログラム構成、画像は営業デモ用のモック表現を含みます。
- 正式な修学旅行見積、旅行契約、申込、行程管理、企業受入可否、個人情報管理はJTB側正式システム・運用で行う前提です。
- 表示価格は参考価格です。正式価格はJTB、参画企業、学校条件、実施人数、行程、講師・会場・運営条件により調整してください。
